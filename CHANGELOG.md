# 更新记录

## 当前状态：已跑通的概念原型

日期：2026-06-12

当前项目已经从一个世界杯预测想法，推进为可以本地运行的概念原型。它还不是完整产品，但核心路径已经打通：用户可以打开本地网页，查看正式分组，录入比分，运行本地预测，配置 OpenAI 兼容 LLM，添加信息源，提取 AI 信息因子，并导出 HTML 报告。

### 已具备能力

- 本地网页服务和启动脚本。
- 2026 世界杯正式分组数据。
- 完整国际比赛历史数据 `data/results.csv`。
- 基于 Elo、Poisson 进球模型和 Monte Carlo 模拟的本地预测。
- 小组赛比分录入和本地保存。
- 模拟次数可配置，默认 `1000` 次。
- 中文队名展示。
- OpenAI 兼容 LLM API 配置和连接测试。
- SSL 证书验证提示和显式跳过选项。
- AI 信息源管理，支持 URL 或手动粘贴正文。
- LLM 提取结构化 AI 信息因子。
- AI 信息因子通过固定规则转成有限 Elo 调整。
- 图形化 HTML 报告导出。
- 项目规则、开发流程和 TODO 文档。

### 当前定位

- 未配置 LLM 时：应用根据完整历史数据、正式分组、已录入比分和本地统计模型生成预测。
- 配置 LLM 后：应用升级为 AI 辅助预测助手，支持信息源解析、AI 信息因子提取和预测报告生成。

### 已知限制

- 仍是本地单用户概念原型，尚未支持网站化部署和多用户隔离。
- 赛程表、近期赛程、已完赛结果浏览尚未实现。
- 小组赛出线预测和按阶段组织预测卡片尚未实现。
- AI 辅助比分录入尚未实现。
- 球队介绍、最近历史战绩、当家明星和身价信息尚未实现。
- 本地预测模型参数仍需系统复盘，包括东道主加成、K 值、赛事权重和回测。
- README 仍需进一步同步模型解释和当前产品边界。

### 下一步方向

- 展示已添加信息源正文，提升 AI 信息因子的可审查性。
- 增加小组赛出线预测。
- 按比赛阶段组织预测卡片。
- 讨论并优化本地预测模型。
- 为网站化部署和多用户版本做架构设计。

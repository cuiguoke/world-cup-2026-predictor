LLM API 设置

用于后续读取权威信息源和生成 AI 解读。API Key 默认只保存在当前浏览器会话。

DeepSeek 示例：Base URL 填 https://api.deepseek.com，模型名可填 deepseek-v4-flash 或 deepseek-v4-pro。

如果测试连接提示“无法确认 HTTPS 服务的身份”，通常不是 API Key 错误，而是本机无法识别服务证书。 普通使用时请先核对 Base URL 是否来自官方；只有在你确认服务可信时，再勾选上方选项重试。
